============
Installation
============


Stable release
--------------

To install derex.builder, run this command in your terminal:

.. code-block:: console

    $ pip install derex.builder

This is the preferred method to install derex.builder, as it
will always install the most recent stable release.

If you don't have `pip <https://pip.pypa.io>`_ installed, this
`Python installation guide <https://docs.python-guide.org/starting/installation/>`_
can guide you through the process.

From sources
------------

The sources for derex.builder can be downloaded from the
`Github repo <https://github.com/silviot/derex.builder>`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/silviot/derex.builder

Or download the `tarball <https://github.com/silviot/derex.builder/tarball/master>`_:

.. code-block:: console

    $ curl  -OL https://github.com/silviot/derex.builder/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install
